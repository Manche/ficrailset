# 架空鉄道まんちぇセット
# fictions vehicle set
simutrans pak64用の架空鉄道車両のセットです。

# 導入方法
testificate版・個別配布版の場合は各形式のフォルダ内のpak
release版の場合はcompositeフォルダ内のpakを導入してください。
